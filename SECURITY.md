# Security Policy

## Supported Versions


| Version | Supported          |
| ------- | ------------------ |
| 0.6.x   | :white_check_mark: |


## Reporting a Vulnerability

To report a security vulnerability:

1. Open an issue in the repository
2. Tag it with `Vulnerability`
3. Include the following:
   - Steps to reproduce
   - Expected behavior
   - Screenshots (if applicable)
   - Environment details
